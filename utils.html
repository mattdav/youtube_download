bin.utils.download_url(video_url: str, music_path: str) → None

Download the song of the youtube video and store it to the music directory as defined in the config file

Parameters:

  • video_url (str) – Url of the youtube video

  • music_path (str) – Path to the user’s music directory

bin.utils.get_music_path(config_path: str) → str

Get the path to the user’s music directory as defined in the config.cfg file If the directory doesn’t exist, return the home’s path instead

Parameters:

config_path (str) – Path to the config folder where the config.cfg file is located

Raises:

  • e – Raise a FileNotFoundError is the config.cfg file doesn’t exist

  • e – Raise a KeyError if the config.cfg file is not correctly filled

Returns:

Return either the path read from the config.cfg file or the home path

Return type:

str

bin.utils.validate_url(url: str) → bool

Check if the input looks like a valid url from youtube via regex

Parameters:

url (str) – input url wrote in the input file music_list.txt

Returns:

True/False

Return type:

bool

>>> validate_url("https://www.youtube.com/watch?v=dQw4w9WgXcQ")
True
>>> validate_url("https://www.invalidurl.com/watch?v=dQw4w9WgXcQ")
False
>>> validate_url("just some random text")
False
